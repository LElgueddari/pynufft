pynufft
=======

pynufft: A pythonic non-uniform fast Fourier transform (NUFFT)

Installation:
-------------

pip install pynufft

Example:
--------

Inside the python environment, type:

            | import pynufft.pynufft
            | pynufft.pynufft.test\_1D() # test an 1-D case
            | pynufft.pynufft.test\_2D() # test the 2D case
